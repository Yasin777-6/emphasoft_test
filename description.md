Тестовое задание 
на позицию 
junior backend developer (python) 
Приложение для бронирования комнат в отеле 
Требования: 
o Для комнат должны быть поля: номер/название, стоимость за сутки, количество 
мест. 
o Пользователи должны уметь фильтровать и сортировать комнаты по цене, по 
количеству мест. 
o Пользователи должны уметь искать свободные комнаты в заданном временном 
интервале. 
o Пользователи должны уметь забронировать свободную комнату. 
o Суперюзер должен уметь добавлять/удалять/редактировать комнаты и 
редактировать записи о бронях через админ панель Django. 
o Брони могут быть отменены как самим юзером, так и суперюзером. 
o Пользователи должны уметь регистрироваться и авторизовываться (логиниться). 
o Чтобы забронировать комнату пользователи должны быть авторизованными. 
Просматривать комнаты можно без логина. Авторизованные пользователи должны 
видеть свои брони. 
Стек:  
o Django; 
o DRF; 
o СУБД предпочтительно PostgreSQL, но не обязательно. Главное не SQLite; 
o При необходимости можно добавлять другие библиотеки. 
Приветствуется: 
o Автотесты; 
o Аннотации типов; 
o Линтер; 
o Автоформатирование кода; 
o Документация к API; 
o Инструкция по запуску приложения.
